01–03 — Lessons 1–3

Right Triangle Trigonometry

  • SOH-CAH-TOA:
  • sin(θ) = opposite/hypotenuse
  • cos(θ) = adjacent/hypotenuse
  • tan(θ) = opposite/adjacent
  • Pythagorean identity: sin²(θ) + cos²(θ) = 1.
  • Inverse trig: θ = sin⁻¹(x) finds the angle.
In a right triangle, if the opposite side = 3 and hypotenuse = 5: sin(θ) = 3/5 = 0.6.

04–05 — Lessons 4–5

Radian Measure

Radians are an alternative unit for measuring angles based on arc length.

  • 1 radian: angle where arc length = radius.
  • π radians = 180°
  • To convert: degrees × π/180 = radians.
  • To convert: radians × 180/π = degrees.
  • Full circle: 2π radians = 360°.
90° = π/2  ·  45° = π/4  ·  60° = π/3  ·  30° = π/6

Topic B: The Unit Circle

Lessons 6–9 · Unit circle coordinates, reference angles, trig values.

06–09 — Lessons 6–9

The Unit Circle

The unit circle has radius 1. Every point (x, y) on it gives cos(θ) = x and sin(θ) = y.

  • Key angles (radians): 0, π/6, π/4, π/3, π/2, π, 3π/2, 2π.
  • Quadrant I: sin > 0, cos > 0.
  • Quadrant II: sin > 0, cos < 0.
  • Quadrant III: sin < 0, cos < 0.
  • Quadrant IV: sin < 0, cos > 0.
  • ASTC: All Students Take Calculus — which trig values are positive.
cos(π/3) = 1/2  ·  sin(π/3) = √3/2  ·  tan(π/3) = √3

Topic C: Graphing Trigonometric Functions

Lessons 10–14 · Graphs of sin/cos/tan, amplitude, period, and transformations.

10–12 — Lessons 10–12

Graphs of Sine & Cosine

  • f(x) = A·sin(Bx + C) + D
  • Amplitude = |A|: half the vertical distance from min to max.
  • Period = 2π/B: horizontal length of one full cycle.
  • Phase shift = −C/B: horizontal translation.
  • Vertical shift = D: moves midline up/down.
  • Cosine starts at max; Sine starts at midline.
f(x) = 3sin(2x) → A=3 (amplitude 3), B=2 (period = 2π/2 = π).

13–14 — Lessons 13–14

The Tangent Function

The tangent function has a period of π and vertical asymptotes where cos(θ) = 0.

  • tan(θ) = sin(θ)/cos(θ)
  • Period: π.
  • Vertical asymptotes: x = π/2 + nπ for integers n.
  • Increases from −∞ to ∞ on each period.
  • No amplitude — unbounded.
tan(0) = 0  ·  tan(π/4) = 1  ·  tan(π/2) is undefined.
