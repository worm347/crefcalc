01 — Lesson 1

Multiple Representations of Linear Functions

A linear function can be expressed as a graph, table, equation, or contextual description — each highlights different features.

  • Graph: shows slope and intercepts visually.
  • Table: constant rate of change between each row.
  • Equation (y = mx + b): slope m, y-intercept b.
  • Context: identifies meaning of variables and rate.
Example: f(x) = 3x + 5 → slope 3 (rises 3 per unit), y-intercept 5 (starts at 5).

02 — Lesson 2

Inverse of a Linear Function

The inverse swaps input and output. If f(a) = b, then f⁻¹(b) = a.

  • To find the inverse: swap x and y, then solve for y.
  • Notation: f⁻¹(x) — read "f inverse of x."
  • Graph: the inverse is a reflection over y = x.
  • Context: if f converts miles to km, f⁻¹ converts km back to miles.
f(x) = 2x + 4 → swap: x = 2y + 4 → solve: y = (x−4)/2 → f⁻¹(x) = (x−4)/2

03 — Lesson 3

Domain Constraints in Context

Real-world situations limit the valid inputs of a function — these are domain constraints.

  • Constraint: a restriction on domain or range based on context.
  • You can't have negative hours, negative prices, or more than 100%.
  • Write constraints as inequalities: 0 ≤ x ≤ 40.
  • Extraneous solutions: algebraic answers that violate constraints.
A worker earns $15/hr: f(h) = 15h. Constraint: 0 ≤ h ≤ 40 (can't work negative or infinite hours).

04 — Lesson 4

Composition of Functions

Composing functions means using the output of one function as the input of another.

  • Notation: (f ∘ g)(x) = f(g(x)) — apply g first, then f.
  • Order matters: f(g(x)) ≠ g(f(x)) in general.
  • Context: a tip calculated on a discounted price — discount first, then tip.
f(x) = 2x, g(x) = x + 3
(f ∘ g)(x) = f(x+3) = 2(x+3) = 2x + 6

05 — Lesson 5

Systems of Two Linear Equations

A system is two or more equations solved simultaneously. The solution is where they intersect.

  • Substitution: isolate one variable, plug into the other equation.
  • Elimination: add or subtract equations to cancel a variable.
  • Graphically: the intersection point is the solution.
  • No solution: parallel lines. Infinite solutions: same line.
2x + y = 8 and x − y = 1 → Add: 3x = 9 → x = 3 → y = 2. Solution: (3, 2)

06 — Lesson 6

Systems with Three Variables

Extend systems to three equations and three unknowns. Solve by eliminating one variable at a time.

  • Step 1: Use two pairs of equations to eliminate the same variable.
  • Step 2: Solve the resulting 2-variable system.
  • Step 3: Back-substitute to find the third variable.
  • Solution is an ordered triple: (x, y, z).
The solution to a 3-variable system represents a point where three planes intersect in 3D space.

07–09 — Lessons 7–9

Absolute Value Functions & Systems

Absolute value creates a V-shaped graph. It can appear in functions and in systems.

  • |x|: returns the non-negative distance from zero. |−3| = 3.
  • f(x) = |x − h| + k: vertex at (h, k), opens upward.
  • Solving |x| = a: gives x = a or x = −a (two cases).
  • System with abs. value: find where |expression| intersects another function.
|x − 2| = 5 → x − 2 = 5 or x − 2 = −5 → x = 7 or −3

10 — Lesson 10

Defining Piecewise Functions

A piecewise function uses different rules for different intervals of the domain.

  • Each "piece" is a separate function valid only on a specified interval.
  • Notation: written with a brace { listing each rule and its domain interval.
  • Evaluating: identify which interval x falls in, then use that rule.
  • The pieces happen in sequence — not simultaneously like a system.
f(x) = { 2x if x < 3 ; x + 5 if x ≥ 3 }
f(1) = 2(1) = 2  ·  f(4) = 4 + 5 = 9

11–13 — Lessons 11–13

Analyzing & Modeling with Piecewise Functions

Use piecewise functions to model real situations where the rule changes at a certain threshold.

  • Tax brackets: different rates for different income ranges.
  • Shipping costs: flat rate up to a weight, then per-pound beyond.
  • Continuity: check that pieces "connect" — no gaps or jumps — at breakpoints.
  • Graph: use open/closed circles to show whether endpoints are included.
A phone plan: $30/mo for 0–2 GB, $30 + $10(x−2) for x > 2 GB → piecewise model.
